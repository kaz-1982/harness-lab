# examples — ハーネスの動作見本

このディレクトリは、ハーネスを使うとどんなドキュメントが生成されるかを **5 分で理解する** ための見本です。
案件作業には使いません(案件作業は `案件単位でリポジトリ全体をコピー` した先で実施)。

## sample_case/

小規模アジャイル案件の最小完成版(v0.10〜)。

### 案件設定
- 案件名: 経理 SaaS — 月次決算 売掛金集計の早期化
- 規模: 小 / 開発手法: アジャイル(Scrum 2週間スプリント)
- 業界: IT・SaaS / DDD: 不採用 / テスト戦略: 通常(Phase 3.5 不使用)
- 事前検討資料: なし(Phase 0 スキップ)
- 画面設計(Phase 2 / ClaudeDesign 領域): スコープ外として省略

### 使い方

検証スクリプトを動かして「緑になるドキュメント群」の手触りを確認できます:

```sh
python harness/tools/check.py examples/sample_case
# → ✅ 機械チェックは問題なし
```

TBD ダッシュボードを生成して中身を覗くこともできます:

```sh
python harness/tools/check.py --tbd examples/sample_case
# → examples/sample_case/_tbd_dashboard.md が生成される
```

### 注意

- 各ドキュメントの内容は **学習のため** に作られたもので、本物の経理 SaaS 案件の機密情報ではありません。
- このディレクトリは **読み取り専用の参照** として扱ってください。新規案件はリポジトリ全体をコピーしてから始めます。
