# meta/ — ハーネス開発専用ディレクトリ

このディレクトリは **ハーネス本体(マスター)の開発・改修に関するメタ情報** を置く場所です。

## 案件コピーには含まれない

`bash harness/tools/new-project.sh <案件パス>` で案件用にコピーするとき、
このディレクトリは **自動的に除外** されます(`new-project.sh` の `rm -rf` 対象)。
案件作業に必要な情報はここに置かないでください。

## 何を置くか

- ハーネスの次バージョン改修バックログ(例: `v0.12-backlog.md`)
- ハーネス改修セッションの引き継ぎログ(`output/_session_log.md` は案件用なので使わない)
- リファクタリング計画・設計メモ
- 改修議論の要約

## 何を置かないか

- 案件作業の成果物 → `output/` 配下に置く
- 案件向けテンプレ → `harness/templates/` に置く
- 案件向け設定 → `output/project_profile.md` に書く
- 学習用サンプル案件 → `examples/sample_case/` に置く(これも案件コピーから除外される)

## 関連

- `examples/` も同様に案件コピーから除外される(学習用 / 案件作業には不要)
- 案件コピーから除外される対象一覧は `harness/tools/new-project.sh` を参照
